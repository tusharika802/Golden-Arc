# Golden ARC Institute - AI-Powered Career Mentorship Platform

## About

Golden ARC Institute is an AI-powered educational platform featuring **Aryam**, an intelligent mentor that helps students and professionals transform their careers through:

- **Upskill**: Precision learning with Socratic questioning
- **Uplift**: Emotional intelligence and empathetic support
- **Unleash**: Agentic coaching with goal tracking and accountability

## Project Overview

This platform provides a comprehensive mentorship ecosystem where users can interact with Aryam through voice and text interfaces, receive personalized career guidance, and track their progress toward their goals.

## Technologies

This project is built with:

- **Vite** - Fast build tool and dev server
- **TypeScript** - Type-safe JavaScript
- **React** - UI library
- **shadcn-ui** - Component library
- **Tailwind CSS** - Utility-first CSS framework
- **React Router** - Client-side routing
- **React Hook Form** - Form management
- **Zod** - Schema validation

## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd path-finder-ai-main

# Step 3: Install dependencies
npm install

# Step 4: Start the development server
npm run dev
```

The application will be available at `http://localhost:8080`

### Build for Production

```sh
npm run build
```

The production build will be in the `dist` directory.

### Preview Production Build

```sh
npm run preview
```

## Features

- **Authentication**: Sign up and log in with email/password
